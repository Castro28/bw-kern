List of Keyboard Shortcuts

Basic Shortcuts

  • Copy: ctrl/cmd + c
  • Paste: ctrl/cmd + v
  • Cut: ctrl/cmd + x
  • Save: ctrl/cmd + s
  • Undo: ctrl/cmd + z
  • (Windows)Redo: ctrl + y
  • (Mac)Redo: cmd + shift + z
  • Find: ctrl/cmd + f
  • Select All: ctrl/cmd + a
  • (Windows) Windows logo key + Shift + S
  • (Mac) cmd + space bar

VSCode Shortcuts

  • Comment: ctrl/cmd + /
  • Move line up or down: alt/option + up/down

VSCode Emmet Abbreviation Shortcuts

  • Doctype template: html:5/!
  • Lorem words: lorem#
  • Split editor: ctrl/cmd + \
